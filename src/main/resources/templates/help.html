Service REST - Aide

Opérations disponibles

1. Insérer un flux

  • URL : /rss25SB/insert
  • Méthode : POST
  • Description : Insère un flux XML rss25SB valide. Rejet si doublon (titre + date).
  • Retour : retourner le status INSERTED avec l’id du dernier item si la requête réussi ou retourner le status ERROR en cas d'erreur

2. Supprimer un article

  • URL : /rss25SB/delete/{id}
  • Méthode : DELETE
  • Description : Supprime l’article par id. Supprime aussi le flux si c’était le dernier article.
  • Retour : retourner le status DELETED si l'article a été supprimé ou le status ERROR en cas d'erreur

3. Afficher un article

  • URL : /rss25SB/resume/xml/{id}
  • Méthode : GET
  • Description : Retourne un article complet au format XML rss25SB
  • Retour : Retourner un article au format XML ou retourner le status ERROR en cas d'erreur

4. Afficher un article (page html)

  • URL : /rss25SB/html/{id}
  • Méthode : GET
  • Description : Retourne un article complet au format XML rss25SB
  • Retour : Retourner un article au format html ou retourner le status ERROR en cas d'erreur

5. Lister les articles

  • URL : /rss25SB/resume/xml
  • Méthode : GET
  • Description : Liste tous les items en résumé (id, date, guid)
  • Retour : XML de la liste des items en résumé

6. Lister les articles (page html)

  • URL : /rss25SB/resume/html
  • Méthode : GET
  • Description : Liste tous les items en résumé (id, date, guid)
  • Retour : Une page html contenant la liste des items en résumé

7. Rechercher des articles

  • URL : /rss25SB/search?date={YYYY-MM-DD}&category={CATEGORIE}
  • Méthode : GET
  • Description : Recherche des articles par date (date postérieure ou égale) ou catégorie
  • Retour : XML

8. Aide

  • URL : /rss25SB/help
  • Méthode : GET
  • Description : Affiche cette page d’aide HTML.
